## PROYECTO PYTHON



#### Resolución de las katas



Abrir un nuevo fichero .py en Visual Studio. En el menú File, seleccionar Save As. Grabarlo como fichero .py con el nombre "Proyecto\_Python". En este fichero está el código de cada kata propuesta de forma consecutiva. Antes de cada código se ha puesto como comentario en número de la consulta y el enunciado. El fichero .py se ha añadido al repositorio de la resolución del proyecto.
